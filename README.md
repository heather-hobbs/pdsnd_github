### Project Title
U.S. Bikeshare Data

### Description
This project calculates descriptive statistics on bikeshare data from Chicago, Washington D.C., and New York City. The user can choose and filter the dataset during analysis.

### Files used
* chicago.csv
* new_york_city.csv
* washington.csv
* bikeshare.py
